MCP Features

  • Graceful degradation if MCP server unavailable
  • 5-second timeout per context fetch with retry logic
  • Context caching to prevent duplicate requests
  • Automatic PII sanitization in log contexts
  • Pattern detection across multiple contexts
  • Provider health monitoring and status display
