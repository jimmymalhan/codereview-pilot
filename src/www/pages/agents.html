Agent Capabilities

5 Core Agents with Specific Responsibilities and Constraints

1. Router Incident Classifier
Input
Raw incident report with symptoms, timestamps, and affected services
Output
Incident type classification (Resource Contention, Memory Leak, Config Error, DNS, Concurrency, Deployment)
Constraint
Must classify to known incident types only. Cannot invent new categories without evidence.
Pattern Matching Classification
2. Retriever Evidence Gatherer
Input
Incident classification and timestamp from Router
Output
List of relevant files, logs, and metrics with file:line citations
Constraint
Must cite actual files in repository using file:line format. Never invent file paths or line numbers.
Evidence Verifier MCP Integration
3. Skeptic Theory Challenger
Input
Initial root cause hypothesis with supporting evidence
Output
Alternative theories with equal or competing evidence
Constraint
Must produce materially different theories, not minor variations. Each alternative must cite evidence.
Hallucination Detector Counter-Analysis
4. Verifier Evidence Validator
Input
Root cause hypothesis with evidence citations from Retriever and Skeptic
Output
Verified evidence with file locations, validated schema references, confirmed API endpoints
Constraint
Must block unsupported nouns (invented fields, APIs, functions). All claims must be verifiable.
Evidence Verifier Hallucination Detector Confidence Scorer
5. Critic Quality Gate
Input
Complete output contract: root cause, evidence, fix plan, rollback plan, tests, confidence score
Output
APPROVED or REJECTED with specific blocking issues listed
Constraint
Blocks if confidence < 0.70 or any output contract field is missing. No partial approvals.
Confidence Scorer Quality Gate

Output Contract

Every diagnosis must produce all of these fields to pass the Critic quality gate:

Root Cause
Why the incident occurred
Evidence
File:line citations from repo
Fix Plan
Specific steps to resolve
Rollback Plan
How to undo the fix
Tests
How to verify the fix
Confidence
Score >= 0.70 required
