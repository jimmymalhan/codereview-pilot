🔄 4-Agent Pipeline

Stage 1: Router - Classify

Input: Your incident description

Output: Failure classification (schema drift, write conflict, stale read, bad deploy, auth, dependency)

Stage 2: Retriever - Gather Evidence

Input: Incident + router classification

Output: Exact evidence with file:line citations

Stage 3: Skeptic - Challenge

Input: Incident + router + retriever

Output: Alternative diagnosis from different failure family

Stage 4: Verifier - Validate

Input: All prior outputs

Output: Final verified diagnosis with confidence score

🛡️ Error Handling

Production-ready error handling for all scenarios

Error Scenarios Handled

💳 No API Credits

When: API key has insufficient credits

UI Response: "💳 No API Credits" + link to add credits

Recovery: Click link, add credits, retry

🌐 Network Errors

When: Server unreachable

UI Response: "🌐 Connection Error"

Recovery: Automatic retry with exponential backoff

⏱️ Request Timeout

When: Diagnosis takes >60 seconds

UI Response: "⏱️ Request Timeout"

Recovery: Automatic retry (up to 2 times)

⚠️ Invalid Input

When: Incident description missing or invalid

Validation Rules:

  • Minimum: 10 characters
  • Maximum: 2000 characters
  • Required: non-empty text

❌ Incomplete Response

When: Server returns partial data

Detection: Missing pipeline stages

Recovery: Automatic retry

⚙️ Server Error (5xx)

When: Server internal error

Recovery: Automatic retry with exponential backoff

🔄 Retry Logic

Attempt 1: Immediate (0 seconds) Attempt 2: 1 second delay Attempt 3: 2 second delay Total: 3 attempts, automatic recovery

Max Retries: 2 (automatic)

Timeout: 60 seconds per request

🚀 Getting Started

Setup and first diagnosis in 4 steps

Step 1: Install

npm install

Installs all dependencies (Anthropic SDK, testing frameworks)

Step 2: Add API Key

export ANTHROPIC_API_KEY=sk-ant-...

Get your API key from console.anthropic.com

⚠️ Important: Your API key must have available credits. Add credits at https://console.anthropic.com/account/billing/overview

Step 3: Start Server

npm start

Starts server at http://localhost:3000 (you're already here!)

Step 4: Submit Incident

  1. Click the "🎯 Diagnose" tab above
  2. Describe your backend failure in the form
  3. Click "Diagnose with Paperclip"
  4. Watch the 4-agent pipeline run in real-time (16-24 seconds)
  5. Review diagnosis with evidence, fix plan, and rollback
